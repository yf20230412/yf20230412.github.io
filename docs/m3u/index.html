使用说明

默认转换为带分组标签的M3U格式，如果不需要，自行取消勾选 group-title 后再执行；
由于各个软件解析M3U列表的方式不同，暂时先考虑如央视卫视等知名电视台的台标；

普通模式：生成的M3U扩展标签内默认无内容。
加强模式：后续根据需求适当匹配内容，诸如匹配台标图片地址等。

扩展参数需要播放器甚至EPG列表配合才能显示它的价值，闭门造车毫无用处，尽量提供多种选项适配市面上流行的客户端。

通用格式举例：

天津
滨海新闻,http://60.30.52.41/live/bhtv1/playlist.m3u8
西青新闻,http://221.238.209.44:81/hls/live1.m3u8
央卫
北京卫视,http://live.zjmc.tv/1/sd/live.m3u8
CCTV-13新闻,https://live.cctv.com/2/sd/live.m3u8

或者以下这种形式也可以直接转为M3U格式

天津,#genre#
滨海生活,http://60.30.52.41/live/bhtv2/playlist.m3u8
西青新闻,http://221.238.209.44:81/hls/live1.m3u8
央卫,#genre#
北京卫视,http://live.zjmc.tv/1/sd/live.m3u8
CCTV-13新闻,https://live.cctv.com/2/sd/live.m3u8

Tip：本地转换不上传不保存任何数据；没有异常处理，对格式要求比较严格；
